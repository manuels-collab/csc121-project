Module 2: Hypertext Markup Language (HTML)

Building the Structure of Web Pages

Introduction to HTML

HTML (Hypertext Markup Language) is the standard markup language used to create web pages. It provides the structure and content for all websites. HTML works hand-in-hand with CSS for styling and JavaScript for interactivity.

Learning Outcomes

After completing this module, you will be able to:

  • Understand the structure of HTML documents
  • Use semantic HTML5 elements appropriately
  • Create forms with various input types
  • Implement accessibility best practices
  • Optimize HTML for search engines (SEO)
  • Debug and validate HTML code

Essential HTML Elements

Element Purpose Example
<h1> - <h6> Headings (6 levels) <h1>Main Title</h1>
<p> Paragraph text <p>This is a paragraph.</p>
<a> Hyperlinks <a href="page.html">Link</a>
<img> Images <img src="image.jpg" alt="Image">
<ul>, <ol>, <li> Lists (unordered/ordered) <ul><li>Item</li></ul>
<form>, <input> Form elements <input type="text" name="username">

HTML Document Structure

Every HTML document follows a basic structure:

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
    <meta charset="UTF-8">
  </head>
  <body>
    <h1>Content goes here</h1>
  </body>
</html>

Semantic HTML5 Elements

Semantic elements clearly describe their meaning to both the browser and developer:

  • <header> - Introductory content
  • <nav> - Navigation links
  • <main> - Main content of the page
  • <article> - Independent, self-contained content
  • <section> - Thematic grouping of content
  • <aside> - Sidebar or related content
  • <footer> - Footer of page or section

HTML Forms

Forms allow users to input data. Here are common form elements:

Input Type Purpose Example
text Single line text input <input type="text">
password Hidden text input <input type="password">
email Email validation <input type="email">
number Numeric input <input type="number">
checkbox Multiple selections <input type="checkbox">
radio Single selection <input type="radio">
submit Submit button <input type="submit">

HTML Best Practices

Use Semantic Elements

Use semantic HTML5 elements to improve accessibility and SEO.

Valid HTML

Always validate your HTML using W3C Markup Validation Service.

Accessibility

Use alt text for images, proper heading hierarchy, and ARIA labels.

Meta Tags

Include meta tags for charset, viewport, and description.
