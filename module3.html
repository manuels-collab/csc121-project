Module 3: Cascading Stylesheets (CSS)

Styling and Designing Beautiful Web Pages

Introduction to CSS

CSS (Cascading Stylesheets) is used to style and layout web pages — for example, to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features. This module teaches you how to create beautiful, responsive, and professional-looking websites.

Learning Outcomes

After completing this module, you will be able to:

  • Write CSS rules and understand selectors
  • Understand the CSS Box Model
  • Create responsive layouts with Flexbox and Grid
  • Implement responsive design principles
  • Use CSS for animations and transitions
  • Apply design best practices and accessibility

CSS Basics

CSS rules consist of a selector and a declaration block:

selector {
  property: value;
  property: value;
}

Example:

h1 {
  color: #333;
  font-size: 32px;
  margin-bottom: 20px;
}

CSS Selectors

Selector Type Description Example
Element Selects all elements of a type p { color: blue; }
Class Selects elements with a specific class .highlight { color: yellow; }
ID Selects element with unique ID #header { background: navy; }
Descendant Selects nested elements ul li { list-style: none; }
Pseudo-class Selects element in specific state a:hover { color: red; }
Attribute Selects elements with attributes input[type="text"] { border: 1px solid; }

The CSS Box Model

Every element in CSS is a box. Understanding the box model is crucial for controlling layout:

  • Content: The actual content (text, images, etc.)
  • Padding: Space inside the element, around the content
  • Border: The edge of the element
  • Margin: Space outside the element, between other elements

Box Model Properties:

div {
  width: 300px;
  padding: 20px;
  border: 2px solid black;
  margin: 10px;
}

Modern Layout Techniques

Flexbox

One-dimensional layout system perfect for navigation menus, card layouts, and centering content.

CSS Grid

Two-dimensional layout system ideal for complex page layouts and dashboard designs.

Media Queries

Responsive design using breakpoints for different screen sizes.

Positioning

Relative, absolute, fixed, and sticky positioning for precise control.

Responsive Design

Create designs that work on all devices using responsive techniques:

Fluid Layout:

/* Use percentages instead of fixed pixels */
.container {
  width: 90%;
  max-width: 1200px;
  margin: 0 auto;
}

Mobile-First Approach:

/* Start with mobile styles, then add complexity */
@media (min-width: 768px) {
  .container {
    width: 80%;
  }
}

@media (min-width: 1024px) {
  .container {
    width: 70%;
  }
}

Colors and Typography

Property Purpose Example
color Text color color: #333;
background-color Background color background-color: #f0f0f0;
font-family Font typeface font-family: Arial, sans-serif;
font-size Text size font-size: 16px;
font-weight Text boldness font-weight: bold;
line-height Line spacing line-height: 1.6;

Animations and Transitions

Transitions:

a {
  color: blue;
  transition: color 0.3s ease;
}

a:hover {
  color: red;
}

Keyframe Animations:

@keyframes slideIn {
  from {
    transform: translateX(-100%);
  }
  to {
    transform: translateX(0);
  }
}

.element {
  animation: slideIn 1s ease-in-out;
}

CSS Best Practices

DRY Principle

Don't Repeat Yourself - Use classes and variables to avoid code duplication.

Mobile-First

Design for mobile devices first, then enhance for larger screens.

Performance

Minimize CSS, use shorthand properties, and optimize images.

Accessibility

Ensure sufficient color contrast and readable font sizes.
